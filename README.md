# pixi-napari-assistant
Workspace to set up napari-assistant for 3D segmentation and regionprops measurements

Created 19.12.2025 for LSM Image Analysis course, Universidad Mayor, Chile.


Install pixi https://pixi.sh/latest/
Create a Pixi folder and run a terminal there.
Run `pixi init napari-assistant`
`cd napari-assistant`

Replace the empty toml file in that directory with the toml file in this repository?

Run `pixi install`

Run `pixi run assistant`
